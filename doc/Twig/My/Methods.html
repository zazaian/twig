my(action=nil) click to toggle source

Provides access to the Twig::My object.

If my hasn’t been called yet, or if action is :clear, :empty or :destroy, a new Twig::My object will be instantiated and all info, favorites, followers and _timeline_objects inside will be deleted. 

# File lib/twig/my.rb, line 84
  def my(action=nil)
    if @my
      case action
      when :empty, :clear, :destroy
        @my = Twig::My.new(@client)
      when nil
        @my
      else
        raise ArgumentError
      end
    else
      @my = Twig::My.new(@client)
    end
  end
