friends(action=nil) click to toggle source

Returns an Array of all recent Twig::Status objects for friends of the Twig::My user. If action is :empty, :clear or :destroy, the Twig::Timeline::Friends object will be re-instantiated, and all fetched Twig::Status objects will be cleared from the box. 

# File lib/twig/timeline.rb, line 35
  def friends(action=nil)
    if @friends
      case action
      when :empty, :clear, :destroy
        @friends = Twig::Timeline::Friends.new(@client)
      when nil
        @friends
      else
        raise ArgumentError, "#{action} is not a valid argument for @active."
      end
    else
      @friends = Twig::Timeline::Friends.new(@client)
    end
  end
